Privacy Policy

Last updated: April 25, 2026

Overview

BaphometsBlade YouTube MCP ("the App") is a personal developer tool that connects to the YouTube Data API on behalf of the authenticated user.

Data Collection

The App does not collect, store, or share any personal data. All data accessed via the YouTube Data API is used solely to fulfill the user's real-time requests and is never stored on any server.

OAuth and Google Account Access

The App uses Google OAuth 2.0 to authenticate users. Access tokens are stored locally on the user's own device and are never transmitted to any third-party server.

The App may access the following YouTube data on your behalf:

  • YouTube channel information
  • Video metadata
  • Playlist data

This data is used only to respond to direct commands from the authenticated user.

Third-Party Services

The App interacts with the YouTube Data API v3 (provided by Google LLC). Google's own privacy policy applies: https://policies.google.com/privacy
